Le développement

Le javascript:

javascript est un langague de programmation de script employé pour rendre une page interactive, le corportement de la page et son apparence pourra varier en fonction de certains paramètres.

le PHP

Le PHP est un langage de programmation libre, utilisé pour géréré des pages web dynamiques,qui sont donc modifiés de certaines informations ( heure, nom de l'utilisateur,formulaire rempli par l'utilisateur etc),

MySQL:

MYSQL est un systeme de gestion de base de données, il permet d'enregistre des information rempli par l'utilisateur (exemple : formulaire de creation de compte) pour ensuite etre ré-utilsé au moment approprié ( ex: connecxion à un site avec identifiant et mot de passe).
